Manual de HTML

Nombre Sintaxis Descripcion
Enlace <a> Se utiliza para crear hipervínculos a otras páginas web o recursos.
Texto enfatizado <strong> Indica que el texto contenido debe ser enfatizado o resaltado, generalmente mostrándolo en negrita.
Párrafo <p> Se usa para estructurar el contenido en párrafos separados.
Imagen <img> Se utiliza para mostrar imágenes en una página web.
Encabezado 1 <h1> Representa el encabezado de nivel 1, el más importante, utilizado para títulos principales.
Enpaquetado <div> se utiliza para crear divisiones o contenedores en una página web, que pueden contener otros elementos HTML y se utilizan para aplicar estilos y organizar el diseño.
Lista no ordenada <ol> se utiliza para crear listas ordenadas, donde los elementos de la lista se presentan con números o letras.
Elemento de lista <li> se utiliza para representar elementos individuales en una lista (<ul> o <ol>).
Tabla <table> se utiliza para crear tablas en HTML, que pueden contener filas (<tr>), columnas (<td> y <th>), y datos tabulares.
Formulario <form> se utiliza para crear formularios interactivos que permiten a los usuarios enviar datos al servidor web.
Campo de entrada <input> se utiliza para crear campos de entrada de datos en un formulario, como cajas de texto, botones de radio y casillas de verificación.
Botón <button> se utiliza para crear botones interactivos en una página web, que los usuarios pueden hacer clic para realizar acciones específicas.
Contenedor <span> se utiliza para aplicar estilos o manipular contenido en línea, sin cambiar el flujo normal del texto.
Énfasis <em> se utiliza para resaltar texto en cursiva, indicando que el contenido es enfatizado.
Texto fuerte <strong> se utiliza para resaltar texto en negrita, indicando que el contenido es especialmente importante o relevante.
Abreviatura <abbr> se utiliza para definir una abreviatura o acrónimo, proporcionando una descripción completa cuando el usuario pasa el mouse sobre él.
Cita en bloque <blockquote> se utiliza para citar un fragmento de texto que debe mostrarse en un bloque separado con sangría.
Cita <cite> se utiliza para citar el título de una obra, como un libro, película o artículo.
Código <code> se utiliza para mostrar código fuente o fragmentos de código en un formato monoespaciado.
Texto preformateado <pre> se utiliza para mostrar texto preformateado, manteniendo el formato y los espacios en blanco exactamente como se escribieron.
Marco de inserción <iframe> se utiliza para incrustar contenido de otra página web dentro de la página actual, como videos de YouTube o mapas de Google.
Audio <audio> se utiliza para insertar archivos de audio en una página web, permitiendo a los usuarios escuchar el contenido multimedia.
Video <video> se utiliza para insertar archivos de video en una página web, permitiendo a los usuarios ver el contenido multimedia.
