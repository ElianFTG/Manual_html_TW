Manual de HTML

Elemento Descripcion
<base> Define una URL base para todas las URL relativas en una página web.
<link> Se utiliza para vincular recursos externos, como hojas de estilo CSS, fuentes o íconos, con la página HTML.
<meta> Proporciona metadatos sobre el documento HTML, como la codificación de caracteres, la descripción de la página y las etiquetas de autor.
<noscript> Contiene contenido alternativo que se muestra si el navegador no admite scripts o si los scripts están deshabilitados.
<script> Se utiliza para incorporar scripts, generalmente JavaScript, en una página web para agregar interactividad y funcionalidad.
<style> Define estilos CSS directamente en el documento HTML para aplicar estilos a elementos en la página.
<title> Define el título de la página web que se muestra en la barra de título del navegador o en los resultados de búsqueda.
<div> Define un contenedor de división genérico que se utiliza para agrupar y estructurar elementos HTML y aplicar estilos a través de CSS. No tiene un significado semántico propio, pero es esencial para la organización y diseño de una página web.
<a> Se utiliza para crear hipervínculos o enlaces a otras páginas web o recursos.
<abbr> Define una abreviatura o acrónimo y proporciona una descripción completa del mismo.
<address> Se utiliza para proporcionar información de contacto o de ubicación, como direcciones postales o de correo electrónico
<article> Define una sección independiente y autónoma de contenido dentro de una página web, como un artículo o una noticia.
<aside> Define un contenido relacionado o secundario que se encuentra en el margen de la página, como un sidebar o una nota al margen.
<audio> Se utiliza para incorporar contenido de audio, como música o efectos de sonido, en una página web.
<b> Define texto en negrita, aunque se recomienda usar para dar énfasis semántico.
<bdo> Se utiliza para cambiar la dirección del texto en línea, como de izquierda a derecha o de derecha a izquierda.
<bdi> Define una sección de texto que se aísla para formatear y dar formato a un idioma bidireccional, como árabe o hebreo.
<blockquote> Se utiliza para citar un bloque de texto de otra fuente, generalmente con sangría y formato especial.
<br> Se utiliza para crear un salto de línea en el texto o en el contenido de la página, sin iniciar un nuevo párrafo.
<button> Crea un botón interactivo que se puede utilizar para activar acciones en una página web, como enviar un formulario.
<canvas> Define una región de dibujo en la que se pueden crear gráficos, imágenes y animaciones utilizando JavaScript.
<cite> Se utiliza para indicar una referencia o cita de una obra, como un libro, artículo o película.
<code> Define un fragmento de código, como código fuente o comandos, que se muestra en un tipo de fuente monoespaciada.
<data> Define un valor numérico o un valor estadístico que se puede asociar con un elemento para su procesamiento posterior.
<datalist> Define una lista de opciones para un elemento <input> con autocompletado, como una lista desplegable.
<del> Marca un fragmento de texto como eliminado o tachado, generalmente para mostrar cambios en un documento.
<details> Define un elemento que puede ocultar o mostrar contenido adicional, como un acordeón o una lista desplegable.
<dfn> Define un término o una frase que se está definiendo o que se considera importante dentro del contenido.
<dl> Define una lista de descripción, que consta de términos (<dt>) y sus definiciones (<dd>).
<dt> Define un término o un título en una lista de descripción (<dl>), que generalmente se utiliza en combinación con la etiqueta <dd> para proporcionar descripciones de esos términos.
<dd> Define la descripción o el detalle asociado a un término o título definido con la etiqueta <dt> en una lista de descripción (<dl>). Se utiliza para proporcionar información adicional sobre el término.
<em> Aplica énfasis al texto, generalmente mostrándolo en cursiva, para resaltar la importancia del contenido.
<embed> Se utiliza para incrustar contenido multimedia, como vídeos o aplicaciones, dentro de una página web.
<fieldset> Agrupa elementos relacionados en un formulario y define una caja o borde alrededor de ellos.
<figure> Se utiliza para agrupar contenido multimedia, como imágenes, y su descripción (<figcaption>) dentro de un contenedor.
<figcaption> se utiliza en conjunto con la etiqueta <figure> para proporcionar una leyenda o descripción para un elemento de contenido, como una imagen o un gráfico, que se encuentra dentro de la etiqueta <figure>. Es especialmente útil para proporcionar contexto o información adicional sobre la figura.
<footer> Define el pie de página de una sección o de toda la página web, que generalmente contiene información de contacto o enlaces relacionados.
<form> Crea un formulario en el que los usuarios pueden ingresar y enviar datos, como campos de texto, botones y selecciones.
<h1>-<h6> Definen encabezados o títulos de diferentes niveles de importancia en una página web, donde <h1> es el más importante y <h6> el menos.
<header> Define el encabezado de una sección o de toda la página web, que generalmente contiene elementos como logotipos, menús y títulos.
<hgroup> Agrupa varios elementos de encabezado (<h1> a <h6>) juntos cuando tienen un propósito común.
<hr> Crea una línea horizontal que se utiliza para separar secciones de contenido en una página web.
<i> Define texto en cursiva, pero generalmente se utiliza para indicar texto en un contexto diferente al del texto circundante.
<iframe> Se utiliza para incrustar un marco en una página web, que puede contener contenido de otra fuente o página.
<img> Se utiliza para mostrar imágenes en una página web, especificando la ruta de la imagen y descripciones alternativas.
<input> Crea un campo de entrada interactivo en un formulario, que puede ser de diferentes tipos, como texto, contraseña o botón de envío.
<ins> Marca un fragmento de texto como insertado o subrayado, generalmente para mostrar cambios en un documento.
<kbd> Se utiliza para representar la entrada del teclado o las pulsaciones de teclas, generalmente mostrándolas en un estilo monoespaciado.
<label> Se utiliza para asociar un texto descriptivo con un elemento de formulario, mejorando la accesibilidad y la usabilidad.
<main> Define el contenido principal de una página web, que suele ser el área central donde se encuentra la información principal.
<map> Se utiliza para crear un mapa de imagen interactivo en el que áreas específicas de una imagen se pueden vincular a enlaces o acciones.
<mark> Se utiliza para resaltar o marcar partes de texto en el contenido, generalmente con un fondo amarillo para destacarlo.
<math> Se utiliza para incorporar contenido matemático en una página web, como fórmulas matemáticas escritas en notación TeX.
<menu> Define una lista de opciones de menú o comandos, que generalmente se utiliza en aplicaciones web o aplicaciones de navegador.
<meter> Se utiliza para mostrar un valor dentro de un rango específico, como una barra de progreso con un valor asociado.
<nav> Define la navegación principal de una página web, que generalmente contiene enlaces a diferentes secciones o páginas del sitio.
<body> Es uno de los elementos más importantes en un documento HTML. Define el contenido visible de una página web, como texto, imágenes, enlaces y otros elementos. Todo lo que los usuarios ven en una página web se encuentra dentro de la etiqueta.
<q> Se utiliza para citar texto en línea, generalmente con comillas dobles, para indicar una cita breve.
<ruby> Se utiliza para mostrar texto ruby, que es una anotación pequeña que se coloca sobre o al lado de un carácter o palabra para indicar su pronunciación o significado.
<object> Se utiliza para incrustar objetos multimedia, como Flash o videos, en una página web.
<ol> Define una lista ordenada (numerada) en la que los elementos de la lista se enumeran secuencialmente.
<output> Se utiliza para mostrar los resultados de un cálculo o una acción realizada en un formulario.
<p> Define un párrafo de texto, que es una unidad básica de contenido de texto en una página web.
<picture> Se utiliza para mostrar imágenes responsivas y adaptativas, permitiendo cargar diferentes versiones de una imagen según las características del dispositivo.
<pre> Define un bloque de texto preformateado, que se muestra con un espacio y un formato fijo, respetando los espacios y saltos de línea.
<progress> Se utiliza para mostrar el progreso de una tarea o una operación, como una barra de progreso.
<s> Se utiliza para representar texto que ha sido eliminado o tachado, generalmente para mostrar cambios en un documento.
<samp> Se utiliza para representar muestras de salida de programas o código, generalmente mostrándolas en un estilo monoespaciado.
<script> Se utiliza para incorporar scripts, generalmente JavaScript, en una página web para agregar interactividad y funcionalidad.
<section> Define una sección temática en una página web, que generalmente agrupa contenido relacionado.
<select> Crea un elemento de lista desplegable en un formulario, que permite a los usuarios seleccionar una opción de una lista.
<small> Define texto que debe mostrarse en un tamaño de fuente más pequeño que el texto circundante, para indicar que es menos importante.
<span> Crea un contenedor en línea sin formato que se utiliza para aplicar estilos o manipular contenido con CSS o JavaScript.
<strong> Define texto en negrita que indica un énfasis fuerte, generalmente utilizado para resaltar la importancia del contenido.
<sub> Se utiliza para representar texto subíndice, que se coloca debajo de la línea de base del texto circundante.
<sup> Se utiliza para representar texto superíndice, que se coloca por encima de la línea de base del texto circundante.
<svg> Se utiliza para incrustar gráficos vectoriales escalables en una página web, que son imágenes basadas en vectores y no pixeles.
<table> Crea una tabla en una página web, que se utiliza para organizar datos en filas y columnas.
<caption> Se utiliza dentro de un elemento de tabla (<table>) para proporcionar un título o descripción de la tabla. Generalmente se coloca como el primer hijo de <table>.
<thead> Define el encabezado de una tabla, que contiene filas de encabezado <tr> que describen las columnas de la tabla.
<tbody> Define el cuerpo principal de una tabla (<table>), que contiene filas <tr> con datos de la tabla.
<tfoot> Define el pie de una tabla, que contiene filas <tr> con resúmenes o totales de los datos de la tabla.
<tr> Define una fila en una tabla (<table>), que generalmente contiene celdas de datos <td> o celdas de encabezado <th>.
<th> Define una celda de encabezado en una tabla (<table>), que generalmente se coloca en la fila de encabezado <thead> y se utiliza para describir el contenido de las columnas.
<template> Define un fragmento de contenido que no se renderiza de inmediato, sino que se puede clonar y utilizar con JavaScript.
<textarea> Crea un área de texto multilínea en un formulario, que permite a los usuarios ingresar texto más largo.
<time> Se utiliza para representar una fecha o un tiempo, con opciones para indicar su formato y zona horaria.
<u> Se utiliza para subrayar el texto, aunque se recomienda usar otros elementos HTML para indicar énfasis o significado.
<ul> Define una lista desordenada en la que los elementos de la lista no se numeran ni ordenan de manera secuencial.
<var> Se utiliza para representar variables o nombres de variables en el texto, generalmente mostrándolos en un estilo cursiva.
<video> Se utiliza para incorporar contenido de video en una página web, permitiendo a los usuarios reproducir videos en línea.
<wbr> Define un posible punto de corte donde un navegador puede dividir una palabra larga o una URL para ajustarla al ancho del contenedor
<area> Se utiliza en conjunto con la etiqueta <map> para definir áreas sensibles en una imagen, como mapas de imágenes o imágenes con zonas clicables.
<col> se utiliza para definir propiedades de una o varias columnas en una tabla HTML. Puedes especificar atributos como ancho, alineación vertical, color de fondo, etc., para aplicar a columnas específicas o a todas las columnas de una tabla.
<colgroup> se utiliza para agrupar una o más etiquetas <col> en una tabla y aplicar propiedades comunes a esas columnas agrupadas. Esto facilita la aplicación de estilos o atributos a múltiples columnas de manera eficiente.
<dialog> Se utiliza para definir una caja de diálogo modal en una página web. Una caja de diálogo modal es una ventana emergente que aparece encima del contenido principal y requiere la interacción del usuario antes de que pueda continuar interactuando con la página principal.
<head> Se encuentra en la sección inicial de un documento HTML y contiene metainformación sobre la página web, como el título de la página, enlaces a hojas de estilo, metadatos, y más. Esta información no se muestra en la página web en sí misma.
<html> es el elemento raíz de un documento HTML y envuelve todo el contenido de la página web. Define el tipo de documento y proporciona el contexto principal para la página.
<legend> Se utiliza en conjunto con la etiqueta <fieldset> en un formulario HTML para proporcionar una leyenda o título descriptivo para un grupo de controles de formulario relacionados.
<li> Define un elemento de lista en una lista ordenada o una lista desordenada Cada <li> representa un elemento de lista.
<optgroup> Se utiliza dentro de un elemento <select> para agrupar opciones relacionadas en una lista desplegable. Ayuda a organizar las opciones cuando se presentan muchas opciones en un menú select.
<option> Define una opción en un elemento <select<, <datalist< o <optgroup<. Cada <option< representa una elección que un usuario puede seleccionar en una lista desplegable o campo de selección.
<rp> Se utiliza junto con la etiqueta <ruby> para proporcionar una alternativa de texto en caso de que el navegador no admita el marcado ruby. Se muestra antes del texto ruby principal.
<rt> Se utiliza junto con la etiqueta <ruby> para proporcionar el texto ruby principal, que es una anotación de pronunciación o significado en caracteres asiáticos, como kanji.
<summary> se utiliza dentro de un elemento <details> y sirve como el encabezado visible que los usuarios pueden hacer clic para expandir o colapsar el contenido oculto dentro del elemento. Es una forma de proporcionar una vista previa o un título para el contenido oculto.
